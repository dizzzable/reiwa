# 📱 Приложение FLClashX

**FLClashX** — приложение с расширенными возможностями для опытных пользователей.

## 🚀 Как установить

1. Открой личный кабинет → **«Мои устройства»**
2. Выбери **FLClashX** в списке приложений
3. Нажми **«Скачать»**
4. Установи приложение

## 🔑 Как настроить

1. Запусти FLClashX
2. Нажми **«Импорт конфигурации»**
3. Выбери способ импорта:
   - **Через QR-код** — отсканируй код из личного кабинета
   - **Через файл** — скачай конфигурацию в личном кабинете и открой её в приложении
4. После импорта нажми **«Подключиться»**

## ⚙️ Особенности

- Гибкие настройки режимов работы
- Подробная статистика использования
- Выбор сервера из списка
