# 📱 Приложение HAPP

**HAPP** — это наше приложение для удобного использования сервиса.

## 🚀 Как установить

1. Перейди в раздел **«Мои устройства»** в личном кабинете
2. Нажми **«Скачать приложение»**
3. Выбери свою платформу (Android / iOS / Windows / macOS)
4. Следуй инструкциям установщика

## 🔑 Как подключиться

1. Открой приложение HAPP
2. Нажми **«Добавить устройство»**
3. Введи свои данные для входа (логин и пароль из личного кабинета)
4. Готово! Приложение автоматически настроит подключение

## ❓ Частые вопросы

**Приложение говорит «Ошибка подключения»**
Проверь интернет-соединение. Если всё в порядке — перезапусти приложение и попробуй снова.

**Не могу найти кнопку «Добавить устройство»**
Убедись, что у тебя последняя версия приложения. Скачай обновление на сайте.
