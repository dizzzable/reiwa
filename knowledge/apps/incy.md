# 📱 Приложение INCY

**INCY** — лёгкое и быстрое приложение для доступа к сервису.

## 🚀 Как установить

1. Зайди в **«Профиль»** → **«Мои устройства»**
2. Нажми **«Скачать INCY»**
3. Выбери версию для своей системы
4. Установи приложение как обычную программу

## 🔑 Как настроить

1. Открой INCY
2. Авторизуйся — используй те же данные, что и в личном кабинете
3. Приложение само подберёт оптимальные настройки
4. Нажми **«Подключиться»**

## 💡 Советы

- INCY потребляет меньше ресурсов, чем HAPP — отлично подходит для старых устройств
- Работает в фоновом режиме
- Автоматически переподключается при обрыве связи
