# 🔌 Подключение к сервису

## 📡 Как подключиться через приложение

1. **Скачай приложение** из раздела «Мои устройства» в личном кабинете
2. **Установи** его на своё устройство
3. **Авторизуйся** — используй логин и пароль от личного кабинета
4. **Нажми «Подключиться»** — всё готово! 🚀

## 💻 Ручная настройка

Если ты хочешь настроить подключение вручную:

1. Зайди в **«Профиль»** → **«Мои устройства»**
2. Нажми **«Показать данные для подключения»**
3. Скопируй строку подключения
4. Импортируй её в своё приложение

## 🌐 Выбор сервера

В приложении можно выбрать сервер из списка доступных. Рекомендуем выбирать сервер, ближайший к твоему региону — это обеспечит наилучшую скорость.

## ✅ Проверка подключения

После подключения открой любой сайт в браузере — если он загружается, значит всё работает! Если нет — попробуй выбрать другой сервер.
